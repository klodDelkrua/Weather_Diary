**Weather Diary** est une application desktop moderne développée en Python avec la bibliothèque **PySide6**. Elle permet de suivre la météo en temps réel de plusieurs villes simultanément, tout en conservant un historique complet des recherches grâce à une base de données locale.



## 🚀 Fonctionnalités

- **Recherche Instantanée** : Obtenez la météo actuelle de n'importe quelle ville au monde via l'API OpenWeatherMap.
- **Tableau de Bord Dynamique** : Ajoutez vos villes favorites dans un tableau de suivi permanent.
- **Historique Persistant** : Toutes vos recherches sont sauvegardées localement dans une base SQLite.
- **Interface Moderne** : Design soigné avec QSS (Qt Style Sheets), bordures arrondies et headers personnalisés.
- **Calcul de l'Heure Locale** : Affiche l'heure exacte de la ville consultée, peu importe votre fuseau horaire.
